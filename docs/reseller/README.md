# Programa Reseller

Vende un sistema completo de WhatsApp + dashboard a **tus propios clientes**, con tu marca, y quédate con el margen.

## Para quién es

* **Agencias de marketing** que ya tienen clientes y quieren añadir automatización al portafolio
* **Consultores de negocio** que asesoran pymes y quieren ofrecerles infraestructura
* **Desarrolladores freelance / no-code** que quieren vender SaaS sin construirlo de cero
* **Integradores de software** con cartera existente

## Cómo funciona

1. Te apuntas al programa (te aprobamos en función del fit)
2. Pagas un **costo mayorista** mensual por cada cuenta activa que tengas
3. Cobras a tus clientes el precio que tú decidas
4. **Margen es tuyo** — directo a tu cuenta

### Ejemplo

* Tu costo mayorista: ~$200/mes por cuenta
* Tu precio público: $497/mes
* **Margen neto: $297/mes por cliente**

Con 5 clientes = $1,485/mes recurrente para ti.

## Beneficios

* **White-label completo** — tu logo, tus colores, tu nombre. Tus clientes nunca ven Impulsa PR
* **Sub-clientes ilimitados** — cuentas para tus clientes desde un panel central
* **20 industry packs** pre-configurados para cualquier nicho
* **Infraestructura cubierta** (WhatsApp Business API, hosting, BD, backups)
* **Soporte directo** para resellers (WhatsApp/Slack)
* **Plataforma probada** — la misma que usamos nosotros con clientes directos
