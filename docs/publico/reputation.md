# Reputación / Reviews

Sistema para **pedir reseñas** automáticamente a tus clientes y amplificar las buenas.

## Cómo funciona

1. Cliente termina su servicio (ej. cita marcada como "completada")
2. Sistema le manda WhatsApp pidiendo feedback
3. Si la respuesta es **positiva (4-5 estrellas)** → lo redirige a Google Reviews / Yelp / Facebook a dejar reseña pública
4. Si es **negativa (1-3 estrellas)** → captura feedback privadamente y te avisa para que tú resuelvas

> Esto multiplica reviews positivas y atrapa quejas antes de que se hagan públicas.

## Activar

Settings → Reputation:

1. Activa toggle
2. Conecta tus links públicos:
   * Google Business Profile
   * Yelp (opcional)
   * Facebook (opcional)
3. Define cuándo se manda el pedido (ej. 24h después de cita completada)
4. Personaliza mensajes (ES + EN)

## Mensaje pedido de review

Plantilla por default:

```
¡Hola [nombre]! Esperamos que disfrutaras tu visita a [tu negocio].
¿Qué tal estuvo? (Califica 1-5)
```

## Si responde 4-5

```
¡Gracias! ¿Nos ayudas dejando una reseña en Google?
👉 [link a Google Reviews]
```

## Si responde 1-3

```
Lamentamos que no fuera lo esperado. ¿Qué pasó?
[Cliente responde]
[Tú recibes notificación interna]
```

> El cliente NO es redirigido a hacer review pública. Tú resuelves privadamente.

## Tracking de reseñas

* Cuántas reviews pediste
* Cuántas dejaron (CTR)
* Promedio de estrellas
* Quejas internas resueltas

## Buenas prácticas

* No pidas review **inmediatamente** después del servicio (cliente todavía está procesando)
* 24-48h después es óptimo
* Personaliza el mensaje con el nombre y servicio específico
* Responde TODAS las reviews públicas (positivas y negativas)
* Las quejas privadas: respóndelas rápido, ofrece solución, conviértelas en clientes leales
