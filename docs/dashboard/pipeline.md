# Pipeline de ventas

Vista **Kanban** de tus leads moviendose por stages — para entender en qué punto del proceso de venta están.

## Stages por defecto

```
Lead nuevo → En conversación → Calificado → Cita agendada → Cliente
```

Cada stage es una columna. Las tarjetas son leads.

## Cómo mover leads

* **Drag & drop** entre columnas (en desktop)
* En mobile: tap el lead → menú → "Mover a..."
* O por bulk action en la tabla `/leads`

## Personalizar stages

Algunos nichos tienen stages distintos por default:

* **Estética**: Lead nuevo → Pregunta info → Cotizado → Reservó depósito → Cita confirmada → Cliente
* **Inmobiliaria**: Lead nuevo → Interesado → Showing agendado → Pre-aprobado → Cerrado
* **Coaching**: Lead nuevo → Sesión descubrimiento → Paquete vendido → En programa → Graduado

> Para personalizar tus stages: Settings → Pipeline → Editar stages.

## Métricas del pipeline

Arriba de cada columna ves:

* **Cantidad** de leads en ese stage
* **Tiempo promedio** que pasan antes de avanzar
* **Tasa de conversión** al siguiente stage

## Atajos

* Click derecho en una tarjeta → menú rápido (mover, etiquetar, abrir conversación)
* Doble click → abre la ficha del lead
