# Tareas (Tasks)

Lista de **cosas pendientes** vinculadas a leads/clientes. Sirve para que tú o tu equipo no olviden seguimientos.

## Crear una tarea

* Desde un lead/cliente → tab **Tareas** → "Nueva tarea"
* Desde `/tasks` → botón **"+ Nueva tarea"** (te pide a quién vincular)

## Campos

* **Título** (ej. "Llamar a Juan para confirmar visita")
* **Vinculada a** (lead, cliente o ninguno)
* **Asignada a** (tú o algún miembro del equipo)
* **Fecha límite** (opcional)
* **Prioridad** (baja/media/alta)
* **Notas**

## Tabs / filtros

* **Para hoy** — vencen hoy
* **Esta semana** — próximos 7 días
* **Atrasadas** — pasaron de fecha
* **Sin asignar**
* **Asignadas a mí**
* **Completadas**

## Marcar completada

Checkbox al lado del título. Se guarda con timestamp para auditoría.

## Recordatorios

Las tareas con fecha generan notificaciones push en el dashboard:

* 1 hora antes
* Al momento del vencimiento
* Si pasaron 24h sin completarla
